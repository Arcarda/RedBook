## 4. Table of Contents (TOC) Setup

In **Affinity 3.0.2**, use the **Table of Contents Panel** (View → Studio → Table of Contents) while in **Layout Studio**.

**Styles to Include in TOC:**

- `H1-Chapter` (The main chapter titles)
- `H1-Section-Title` (Section I, Section II, etc.)

**TOC Layout Tip:**

- Chapter Title ........................ Page #
- Use "Tab Stops" with a "Leader" (.) to create the dots.

---

## 5. Preface / Introduction

Starts on a **Right Hand Page**. Use Arabic numerals (1, 2, 3) or Roman numerals (ix, x) depending on your preference for front matter.

## 6. Case Brief Styling

Whenever you see a **CASE BRIEF** in the manuscript, consider:

- Setting it in a slightly different font (e.g., Sans Serif)
- Using a 0.5" left indent
- Applying a subtle "Case Brief" character style to the title.
