# The Red Book — Pre-Publishing Checklist

## Before Opening Affinity Publisher

### Files Ready

- [ ] `Red_Book_Manu_FINAL.md` — Complete manuscript (181KB, 4,689 lines)
- [ ] Convert to DOCX for easier import: `pandoc Red_Book_Manu_FINAL.md -o Red_Book_Manu.docx`

### Fonts Installed

- [ ] EB Garamond (Regular, Bold, Italic) — [Download from Google Fonts](https://fonts.google.com/specimen/EB+Garamond)
- [ ] Alternative: Crimson Pro or Libre Baskerville

### Graphics Prepared (Optional)

- [ ] Chapter opener icon (ship's wheel, anchor, or compass) — SVG or PNG
- [ ] Section break ornament

---

## Document Setup Checklist

### Initial Configuration

- [ ] Trim size: 6" × 9"
- [ ] Facing pages: Enabled
- [ ] Color mode: Grayscale (interior) / CMYK (cover)
- [ ] Resolution: 300 DPI
- [ ] Margins configured (Inside: 1", Outside: 0.875", Top: 0.75", Bottom: 0.875")
- [ ] Bleed: 0.125" all sides

### Master Pages Created

- [ ] A-Master: Standard text pages (running headers + page numbers)
- [ ] B-Master: Chapter openers (no headers/footers)
- [ ] C-Master: Section title pages
- [ ] D-Master: Front matter (title, copyright, dedication)

### Text Styles Created

- [ ] Body (11pt, 1.35 leading, 0.25" first-line indent)
- [ ] Body-First (no indent — use after headings)
- [ ] H1-Chapter (28pt bold)
- [ ] H2-Section (14pt bold)
- [ ] Operators-Orders-Title (11pt bold small caps)

---

## Import & Styling Checklist

### Import Manuscript

- [ ] File → Place → Select DOCX file
- [ ] Flow text through linked frames
- [ ] Apply A-Master to body pages
- [ ] Apply B-Master to chapter opener pages

### Apply Formatting

- [ ] Style all chapter titles (H1-Chapter)
- [ ] Style all section headings (H2-Section)
- [ ] Remove first-line indent after headings (Body-First)
- [ ] Create Operator's Orders boxes (19 total — one per chapter)

### Check Layout

- [ ] No overset text (no red + icons)
- [ ] No widows (single words at top of page)
- [ ] No orphans (single lines at bottom of page)
- [ ] Chapter openers start on recto (right) pages

---

## Front Matter Checklist

### Pages to Create

- [ ] Title page (THE RED BOOK)
- [ ] Copyright page (© Year, Author)
- [ ] Dedication page (optional)
- [ ] Table of Contents (auto-generate or manual)
- [ ] Preface: BEFORE WE BEGIN

### Page Numbering

- [ ] Main content: Arabic numerals starting at 1 (from Chapter 0)
- [ ] Section Manager configured correctly

## Cover Checklist

### Gather Information

- [ ] Final page count (for spine width calculation)
- [ ] ISBN (if using your own)
- [ ] Author name as it should appear
- [ ] Printer's cover template downloaded

### Cover Elements

- [ ] Front cover design finalized
- [ ] Back cover with blurb and barcode space
- [ ] Spine with title and author name
- [ ] Full wraparound cover file created

---

## Final Export Checklist

### Before Export

- [ ] Spell check complete
- [ ] All images are 300 DPI
- [ ] Fonts embedded (check panel)
- [ ] Bleeds verified (View → Show Bleed)
- [ ] Page count is even

### Export Settings

- [ ] Format: PDF/X-1a:2003
- [ ] Include bleed: ✓
- [ ] Include marks: ✗ (printer adds these)
- [ ] Color mode matches printer requirements

### Upload

- [ ] Interior PDF uploaded to printer (KDP, IngramSpark, etc.)
- [ ] Cover PDF uploaded
- [ ] Proof copy ordered
